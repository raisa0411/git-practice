
# Lab Summary:

In this lab, we gained practical knowledge of software engineering tools used for project management and version control. We learned how to use Trello to organize software projects by creating boards, lists, and cards to assign tasks, track progress, set deadlines, and collaborate effectively with team members.

 We also learned the fundamentals of Git, a distributed version control system. We practiced essential Git operations, including initializing a repository, checking the repository status, staging changes, committing updates with meaningful commit messages, configuring Git user information, and pushing local commits to a remote GitHub repository. Additionally, we learned how Git tracks project history, supports collaboration, and helps manage different versions of a project.

 Overall, the lab introduced us to industry-standard tools for project planning, team collaboration, and source code management, helping us understand how software development teams efficiently organize work and maintain code throughout the software development lifecycle.
